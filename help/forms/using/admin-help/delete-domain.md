# 删除域{#delete-a-domain}

使用“域管理”页将现有域标记为已过时。

如果创建的域与已删除的域名相同，则会恢复已删除的域及其包含的信息。 无法删除DefaultDom域。

1. 在管理控制台中，单击&#x200B;**[!UICONTROL 设置>用户管理>域管理]**。
1. 选中要删除的域的复选框，单击&#x200B;**[!UICONTROL Delete]**，然后单击&#x200B;**[!UICONTROL OK]**。
