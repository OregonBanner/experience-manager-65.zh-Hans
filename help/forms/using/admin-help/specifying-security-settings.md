# 指定安全性設定 {#specifying-security-settings}

Forms可讓您控制是否解析XML輸入中的外部實體。 依預設，這些問題會得到解決，但您可以變更此行為以提高AEM表單系統的安全性。

**防止處理包含外部實體參照的XML資料檔案**

1. 在管理控制檯中，按一下 **[!UICONTROL 服務> Forms]**.
1. 清除「解析外部圖元」核取方塊。
1. 单击“**[!UICONTROL 保存]**”。
