# 搜索用户或用户组 {#search-for-a-user-or-group}

1. 在管理控制台中，单击“设置”>“用户管理”>“用户和用户组”。
1. 指定信息以缩小搜索范围，然后单击“查找”。 搜索结果将列在页面底部。 您可以通过单击任何列标题对列表进行排序。 最多返回1000个结果。
