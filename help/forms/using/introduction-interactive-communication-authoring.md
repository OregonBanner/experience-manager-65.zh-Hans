# 交互通信创作UI{#introduction-to-interactive-communication-authoring-ui}简介

创作[交互式通信](/help/forms/using/interactive-communications-overview.md)的用户界面是直观的，并提供了以下内容以创作交互式通信的打印和Web渠道:

* WYSIWYG拖放文档编辑器
* 集成的资产存储库 — 上传到服务器并在服务器上创建的资产可在交互式通信创作界面的资产浏览器中使用

当您[创建新的或编辑现有的Interactive Communication](../../forms/using/create-interactive-communication.md)时，您使用以下用户界面元素：

* [侧栏](#sidebar)
* [页面工具栏](#page-toolbar)
* [组件工具栏](#component-toolbar)
* 内容区域

提要栏包含以下内容：

* **渠道浏览器**

渠道浏览器可帮助您在交互通信的打印渠道和Web浏览器之间切换。 根据您在渠道浏览器中选择的渠道，诸如“内容和组件”等浏览器将显示相应选项。

* **内容**
浏览器在内容浏览器中，您可以看到所选渠道的文档的对象层次结构。作者可以通过点按文档对象树中的元素来导航到特定组件。 作者可以在Web渠道中搜索对象，并从此树中重新排列它们。

* **属性浏览器**

   允许您编辑组件的属性。 属性会根据组件而更改。 例如，要查看文档容器的属性：
选择一个组件，然后点按![字段级别](assets/field-level.png) > **文档容器**，然后点按![cmpr](assets/cmppr.png)。

* **资产浏**
览器 — 隔离不同类型的内容，如布局片段、图像、文档、页面、视频。作者可以将资源拖放到交互式通信中。

* **组**
件浏览器包括可用于构建文档的打印和Web渠道的组件。您可以将组件拖动到交互式通信中以添加元素，并根据需要配置添加的元素。 下表介绍了在用于打印和Web渠道的组件浏览器中列出的组件：

| **组件** | **打印渠道** | **Web 渠道** | **功能** |
|---|---|---|---|
| 图表 | ✓ | ✓ | 添加一个图表，您可以在交互式通信中使用该图表，以可视方式表示从表单数据模型集合项检索的二维数据。 |
| 文档片段 | ✓ | ✓ | 允许您向交互式通信中添加可重用的组件、文本、列表或条件。 您添加到交互式通信的可重用组件可以是基于表单数据模型的组件，也可以是没有表单数据模型的组件。 |
| 图像 | ✓ | ✓ | 允许插入图像。 |

| 表 | * | ✓ | 添加表格以便按行和列整理数据。 |
| 目标区域 | ** | ✓ | 在Web渠道中插入一个目标区域，以组织Web渠道特定的组件。 |

* **数据源浏**
览器数据源浏览器以您在创建交互式通信时选择的表单数据模型显示可用数据源。

### 使用组件{#key-points-for-working-with-components}的关键点

使用交互式通信组件时的要点如下：

* 每个组件都具有可控制其外观和功能的关联属性。 要配置组件的属性，请点按组件，然后点按![cmppr](assets/cmppr.png)以在“属性”浏览器中打开组件属性。
* 组件使用其元素名称进行标识。 点按![cmppr](assets/cmppr.png)时，可以通过更改属性浏览器中的元素名称字段值来更改组件的名称。 元素名称字段仅接受字母、数字、连字符(-)和下划线(_)。 不允许使用其他特殊字符，元素名称应以字母开头。
* 只要标题在交互式通信中可见，您就可以在编辑器中内嵌修改交互式通信组件的标题属性，而无需打开属性浏览器。 为此，请执行以下操作：

   1. 点按可选择具有标题属性且其隐藏标题属性被禁用的组件。
   1. 点按![aem_6_3_edit](assets/aem_6_3_edit.png)以使标题可编辑。

   1. 修改标题并点按返回键，或点按组件外的任意位置以保存更改。 点按Esc键可放弃更改。

## 组件工具栏{#component-toolbar}

![组件工具栏标签](do-not-localize/component_toolbar_labels_new.png)

选择组件时，您会看到一个工具栏，通过它可以使用组件。 您可以选择剪切、粘贴、移动和指定组件的属性。 您的选择是：

A.**配置**:点按&#x200B;**配置**&#x200B;时，组件属性显示在提要栏中。

B.**编辑规则**:当您点按编辑规则时，将显示规则编辑器，您可以在其中编辑和创建选定组件的规则。 在规则编辑器中，您还可以选择其他表单对象（组件），并编辑/创建这些表单对象的规则。

C.**复制**:您可以使用复制选项来复制组件并将其粘贴到交互通信中的其他位置。

D.**剪切**:您可以在交互通信中使用剪切选项将组件从一个位置移动到另一个位置。

E.**删除**:允许您从交互式通信中删除组件。

F.**插入组件**:允许您在选定组件上方插入组件。

G.**粘贴**:允许您使用上述选项粘贴您剪切或复制的组件。

H.**组**:如果要剪切、复制或粘贴多个组件，可选择多个组件。

我。**父项**:允许您选择组件的父项。

J.**视图SOM表达式:**&#x200B;允许您视图组件的[ SOM表达式](../../forms/using/using-som-expressions-adaptive-forms.md)。

K:**将面板中的对象分组：**&#x200B;允许您将面板中的组件分组，以便能够同时对这些组件执行操作。 有关详细信息，请参阅Panel](create-interactive-communication.md#groupobjectspanel)中的[对象分组。

L.**添加子面板**（仅适用于面板）：允许您向面板中添加子面板。

M:**添加面板工具栏**（仅适用于面板）：允许您添加面板组件的工具栏。 然后，您可以在工具栏上执行其他操作。

此外，工具栏上的&#x200B;**替换**&#x200B;选项允许您用替代组件替换现有组件。 此选项对面板组件不可用。

## 页面工具栏 {#page-toolbar}

顶部的“页面”工具栏提供了一些选项，用于预览交互式通信并更改其属性。 在创作交互式通信时，您可以预览该通信，并相应地进行更改。 在页面工具栏中，您会看到：

* 切换侧面板![切换侧面板](assets/toggle-side-panel.png):允许您显示或隐藏提要栏。
* 页面信息![页面信息ad](assets/pageinformationad.png):允许您视图页面属性。
* 模拟器![标尺](assets/ruler.png):可让您针对平板电脑和手机等不同的显示尺寸模拟交互式通信的外观。
* 编辑：允许您选择其他模式，例如：编辑、样式、开发人员和设计。

   * 编辑：允许您编辑交互式通信及其组件的属性。 例如，添加组件、删除图像以及指定必填字段。
   * 样式：允许您设置交互通信组件外观的样式。 例如，在样式模式下，您可以选择面板并指定其背景颜色。
   * 开发人员：允许开发人员：

      * 了解交互式通信由哪些内容组成。
      * 在何时何地调试发生的情况，这反过来有助于解决问题。
   * 目标:允许您启用或禁用未列在提要栏中的自定义组件或现成组件。


* 预览:允许您预览发布交互式通信时的外观。
